## Example categories

### Basic usage

Start with these examples to understand the fundamentals:

- **[Basic particles](/examples/basic-particles)** - Learn how to set up your first particle system
- **Particles demo** - See millions of particles in action

### Advanced techniques

Explore more complex implementations:

- **[Transitions](/examples/transitions)** - Animate particles between positions
- **[Shapes](/examples/shapes)** - Create 3D shapes with particles
- **Dynamic instances** - Add and remove instances on the fly

<Tip>
  All examples are open source and available on CodePen. You can fork them and experiment with the code.
</Tip>

## Running examples locally

You can run the demo locally to experiment with the examples:

```bash
npm install
npm start
```

The demo will be available at `http://localhost:8080`.

<Note>
  The demo code is located in `demo/src/index.js` and includes a complete implementation with multiple instances, transitions, and dynamic attributes.
</Note>
